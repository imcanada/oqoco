oqoco

script controlled div pages

about

oqoco is a javascript plugin / script that allows you to use div's as pages. it does this by listening for the navigation link you pressed and take it's 'href' attribute to change the display mode of the div the link points to so that it gets shown while hiding all the other divs. oqoco is licensed under the unlicense placing it into public domain. unlicense.org

use

to use the plugin download the script from github and put somewhere in your website folder and import. in the script file you can change some settings like the classname of your "pages" and the default page (homepage). in order to have pages you have to have multiple div's with each div acting as a page with content in it. the div's must have the classname of your pages defined in the config and an id that your nav links then point to. your nav links can be any clickable element with a href attribute. you can use this demo site available on the github repo as a starting point to get an idea of how to set this up.
